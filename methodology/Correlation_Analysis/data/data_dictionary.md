# Data Dictionary

## Purpose

This document describes the datasets used in the environmental correlation analyses conducted as part of the Master's thesis.

The analyses combine two categories of annual time series:

1. Regional and European recruitment estimates generated by the Glass Eel Recruitment Estimation Model (GEREM).
2. Large-scale environmental variables representing potential drivers of recruitment variability.

The environmental files are **analysis-ready datasets**, rather than untouched raw-source files. Several contain lagged values and multi-year averages created during preparation of the original analyses.

---

# Dataset Overview

| File | Role | Temporal coverage | Main contents |
|------|------|-------------------|---------------|
| `fit7gerem.csv` | Response variables | 1960–2015 | Six regional recruitment estimates and total European recruitment |
| `GSI.csv` | Environmental predictor | 1966–2015 | Gulf Stream Index, lagged values, and lagged multi-year averages |
| `NAO.csv` | Environmental predictor | 1950–2015 | North Atlantic Oscillation index, lagged values, and lagged multi-year averages |
| `SST.csv` | Environmental predictor | 1960–2015 | Sea surface temperature, lagged values, and lagged multi-year averages |
| `PP.csv` | Environmental predictor | 1989–2012 | Primary production index |

The effective analysis period varies among environmental variables according to data availability and the temporal alignment performed in the corresponding R scripts.

# GEREM Recruitment Estimates

## File

```text
fit7gerem.csv
```

## Description

This file contains annual recruitment estimates generated by the Glass Eel Recruitment Estimation Model (GEREM).

It includes recruitment estimates for six European regions and an estimate of total European recruitment.

These estimates constitute the response variables in the environmental correlation analyses.

Rather than correlating environmental conditions directly with individual monitoring series, the analyses use GEREM outputs that integrate information from multiple monitoring programs and observation types.

## Temporal Coverage

```text
1960–2015
```

## Dimensions

```text
56 annual observations
8 columns
```

## Variables

| Variable | Description |
|----------|-------------|
| `Annee` | Calendar year |
| `Rzone1` | Recruitment estimate for the British Isles region |
| `Rzone2` | Recruitment estimate for the Bay of Biscay / French Atlantic region |
| `Rzone3` | Recruitment estimate for the North Sea region |
| `Rzone4` | Recruitment estimate for the English Channel region |
| `Rzone5` | Recruitment estimate for the Iberian Atlantic region |
| `Rzone6` | Recruitment estimate for the Mediterranean region |
| `Rglobal` | Estimated total European recruitment |

## Region Correspondence

| GEREM variable | Region | GEREM zone code |
|----------------|--------|-----------------|
| `Rzone1` | British Isles | `BI` |
| `Rzone2` | Bay of Biscay / French Atlantic | `ATL_F` |
| `Rzone3` | North Sea | `NS` |
| `Rzone4` | English Channel | `Manche` |
| `Rzone5` | Iberian Atlantic | `ATL_IB` |
| `Rzone6` | Mediterranean | `Med` |

## Use in the Analyses

The regional recruitment estimates are generally:

1. log-transformed to reduce skewness and stabilize variance;
2. rescaled before comparison with environmental variables;
3. processed differently depending on the temporal scale of interest.

For long-term analyses, the regional estimates are smoothed using a five-year moving average.

For short-term analyses, their trends and autocorrelation structures are assessed, and differencing or ARIMA modelling is used to obtain approximately stationary residual series.

The European estimate, `Rglobal`, is present in the file but is excluded from several scripts that focus on the six regional recruitment estimates.

# Gulf Stream Index

## File

```text
GSI.csv
```

## Description

This file contains an annual Gulf Stream Index and derived variables representing lagged Gulf Stream conditions.

The Gulf Stream Index was used to investigate whether changes in North Atlantic circulation and Gulf Stream position were associated with subsequent European eel recruitment.

## Temporal Coverage

The file contains rows for:

```text
1966–2015
```

The annual value `ind_GSI` is missing for 2015, while lagged variables derived from previous years remain available for that row. The principal observed annual series therefore extends through 2014.

## Dimensions

```text
50 annual rows
7 columns
```

## Variables

| Variable | Description |
|----------|-------------|
| `annee` | Calendar year |
| `ind_GSI` | Gulf Stream Index for the current year |
| `GSI_1` | Gulf Stream Index one year before the listed year |
| `GSI_2` | Gulf Stream Index two years before the listed year |
| `GSI_3` | Gulf Stream Index three years before the listed year |
| `MOY_GSI_2` | Mean Gulf Stream Index over the preceding one and two years |
| `MOY_GSI_3` | Mean Gulf Stream Index over the preceding one, two, and three years |

## Interpretation of Lagged Variables

For a listed year \(t\):

```text
GSI_1 = GSI at year t - 1
GSI_2 = GSI at year t - 2
GSI_3 = GSI at year t - 3
```

The lagged averages summarize environmental conditions during possible multi-year larval migration periods:

```text
MOY_GSI_2 = mean(GSI at t - 1 and t - 2)

MOY_GSI_3 = mean(GSI at t - 1, t - 2, and t - 3)
```

At the beginning of the series, some averages are based on fewer available lagged observations.

## Use in the Analyses

The GSI dataset was used in both long-term and short-term analyses.

### Long-term analysis

The GSI-derived series and regional recruitment estimates were smoothed using five-year moving averages before Pearson correlations were calculated.

### Short-term analysis

The annual and lagged GSI series were assessed for:

- temporal trends;
- stationarity;
- autocorrelation;
- appropriate ARIMA structure.

Differenced values or ARIMA residuals were then used in cross-correlation analyses with pre-whitened recruitment series.

# North Atlantic Oscillation

## File

```text
NAO.csv
```

## Description

This file contains an annual North Atlantic Oscillation index and derived lagged variables.

The NAO represents large-scale atmospheric variability across the North Atlantic and was used as an integrative indicator of climatic conditions that may affect ocean circulation, temperature, transport, and larval survival.

## Temporal Coverage

```text
1950–2015
```

The short-term scripts generally extract the period overlapping the GEREM estimates:

```text
1960–2015
```

## Dimensions

```text
66 annual observations
7 columns
```

## Variables

| Variable | Description |
|----------|-------------|
| `annee` | Calendar year |
| `ind_NAO` | North Atlantic Oscillation index for the current year |
| `NAO_1` | NAO index one year before the listed year |
| `NAO_2` | NAO index two years before the listed year |
| `NAO_3` | NAO index three years before the listed year |
| `MOY_NAO_2` | Mean NAO index over the preceding one and two years |
| `MOY_NAO_3` | Mean NAO index over the preceding one, two, and three years |

## Interpretation of Lagged Variables

For a listed year \(t\):

```text
NAO_1 = NAO at year t - 1
NAO_2 = NAO at year t - 2
NAO_3 = NAO at year t - 3
```

The lagged averages summarize environmental conditions over possible multi-year exposure or migration periods:

```text
MOY_NAO_2 = mean(NAO at t - 1 and t - 2)

MOY_NAO_3 = mean(NAO at t - 1, t - 2, and t - 3)
```

At the beginning of the series, some averages are calculated from fewer available lagged observations.

## Use in the Analyses

The archived final-analysis script uses the NAO primarily for short-term analyses.

The NAO series and regional recruitment estimates are assessed for:

- deterministic trends;
- stochastic stationarity;
- autocorrelation;
- suitable ARIMA structures.

Where necessary, differencing or ARIMA modelling is applied. The resulting residual or differenced series are then compared using cross-correlation functions.

1. aligning SST and GEREM estimates over 1960–2015;
2. log-transforming regional recruitment;
3. rescaling recruitment and SST variables;
4. applying five-year moving averages;
5. calculating Pearson correlations;
6. adjusting the significance assessment for temporal autocorrelation.

# Primary Production

## File

```text
PP.csv
```

## Description

This file contains an annual primary production index for the Sargasso Sea.

Primary production was used as a proxy for food availability and ecosystem productivity during the early oceanic stage of European eel larvae.

Unlike the GSI, NAO, and SST files, this file contains only the annual index and does not include precomputed lagged variables or moving averages.

## Temporal Coverage

```text
1989–2012
```

## Dimensions

```text
24 annual observations
2 columns
```

## Variables

| Variable | Description |
|----------|-------------|
| `Annee` | Calendar year |
| `ind_PP` | Annual primary production index |

## Use in the Analyses

Primary production was used in the short-term correlation analysis.

The workflow consists of:

1. extracting GEREM recruitment estimates for the same 1989–2012 period;
2. log-transforming regional recruitment estimates;
3. rescaling recruitment and primary production;
4. testing trends and stationarity;
5. applying first differencing or ARIMA models where necessary;
6. examining residual autocorrelation;
7. using cross-correlation functions to assess lagged relationships.

The shorter temporal coverage of this dataset results in a smaller sample size than the analyses involving GSI, NAO, or SST.

# Derived Variables and Analysis-Ready Data

The environmental datasets should be considered **analysis-ready historical files**.

The following derived variables were already present before the R scripts were run:

- one-year lagged values;
- two-year lagged values;
- three-year lagged values;
- averages of the preceding two years;

These variables were created to test environmental conditions corresponding to alternative assumptions about larval migration duration and delayed recruitment responses.

They are preserved because they were directly used in the historical analyses.

# Transformations Applied in the R Scripts

The archived scripts apply additional transformations after importing the datasets.

## Recruitment transformation

GEREM recruitment estimates are log-transformed:

```r
gerem <- log(gerem)
```

This transformation is intended to reduce skewness and stabilize variance.

## Rescaling

The scripts describe the following operation as a z-score or standardization:

```r
(x - mean(x)) * (1 / var(x))
```

This centers each series and divides by its **variance**.

This is not the conventional z-score, which divides by the standard deviation:

```r
(x - mean(x)) / sd(x)
```

The historical operation has been preserved unchanged in the original scripts. In this repository, it is described as **rescaling** rather than conventional standardization to avoid ambiguity.

## Long-term transformations

Long-term scripts additionally apply five-year moving averages using:

```r
forecast::ma(..., order = 5)
```

## Short-term transformations

Short-term scripts may apply:

- first differencing;
- ARIMA modelling;
- extraction of ARIMA residuals;
- pre-whitening;
- cross-correlation analysis.

Further explanations are provided in:

- `documentation/time_series_methods.md`
- `documentation/workflow.md`
- `documentation/code_walkthrough.md`

# Missing Values

Missing values occur primarily where lagged environmental variables cannot be calculated because preceding years are unavailable.

For example:

- the first year of a series cannot contain an internally calculated one-year lag;
- the first two years cannot contain a complete two-year lag history;
- the first three years cannot contain a complete three-year lag history.

Some multi-year means at the beginning of a file are based on fewer available preceding observations.

In `GSI.csv`, the current-year index is also missing for 2015, although lagged values based on earlier years remain present.

The R scripts account for some edge-related missing values by restricting the analysis period or removing rows affected by moving-average calculations.

The datasets differ in temporal coverage.

| GEREM recruitment | 1960–2015 |

Consequently, sample size varies among analyses.

# Relationship Between the Datasets

The data flow can be summarized as:

```text
European recruitment observations
                 │
                 ▼
              GEREM
                 │
                 ▼
Six regional recruitment estimates
                 │
                 ├───────────────┐
                 ▼               ▼
       Long-term analysis  Short-term analysis
                 ▲               ▲
                 │               │
      Environmental time-series datasets
```

GEREM recruitment estimates provide the response variables.

The GSI, NAO, SST, and PP datasets provide environmental predictor variables representing different ecological hypotheses.
