# Reproducibility Notes

## Purpose

This repository serves as a historical research archive documenting the environmental correlation analyses conducted during my Master's research (2015–2016).

The objective is to preserve the original analytical workflow, statistical methodology, and scientific reasoning underlying the study rather than to provide a fully modernized software package.

---

# Historical Archive

The original R scripts have been preserved in their historical form with zero modifications.

---

# Computing Environment

The analyses were conducted approximately ten years before the creation of this repository.

The original computational environment (R version, package versions, operating system, and software dependencies) is no longer fully documented.

Consequently, exact computational reproducibility cannot be guaranteed without reconstructing the historical software environment.

---

# Analysis-Ready Datasets

Several environmental datasets included in this repository were already prepared prior to the archived analyses.

Depending on the dataset, they contain:

- lagged environmental variables;
- moving-average variables;
- derived environmental indices.

These files are preserved exactly as they were used in the original analyses.

Their presence reflects the historical workflow followed during the Master's project.

---

# Original Coding Style

The scripts intentionally preserve the programming style used during the original research project.

Characteristics include:

- repeated code between scripts;
- manual preprocessing steps;
- limited internal documentation;
- sequential analysis workflow.

These characteristics reflect the historical implementation and have been retained to preserve reproducibility.

---

# Repository Philosophy

This repository is intended primarily as a scientific archive.

Its principal value lies in documenting:

- the analytical workflow;
- the statistical methodology;
- the ecological hypotheses;
- the interpretation of the analyses.

It should not be interpreted as an example of modern software engineering practices.

---

# Documentation

To facilitate understanding of the archived analyses, extensive documentation accompanies the original code.

The documentation includes:

- scientific rationale;
- analytical workflow;
- statistical methods;
- assumptions;
- data sources;
- code walkthroughs.

These documents provide the context necessary to understand the original implementation without modifying the archived scripts.

---

# Reproducing the Analyses Today

In principle, the analyses could be reproduced using the archived datasets and scripts.

However, successful execution may require:

- adapting file paths;
- installing archived R packages or compatible modern equivalents;
- updating deprecated package functions where necessary;
- reconstructing portions of the historical computing environment.

Such adaptations would represent technical updates only and should not alter the underlying analytical methodology.

---

# Relationship to the Thesis

The analyses documented in this repository correspond to the environmental correlation chapter of the Master's thesis.

Together with the Dynamic Factor Analysis (DFA) and Glass Eel Recruitment Estimation Model (GEREM) repositories, they form the complete quantitative framework developed during the project.

The three repositories document successive stages of the research:

1. Characterization of recruitment dynamics (DFA).
2. Estimation of regional recruitment (GEREM).
3. Investigation of environmental drivers (Correlation Analysis).

---

# Future Development

No further development of the archived code is planned.

Should these methods be revisited in future research, a modern implementation would preferably be developed independently rather than modifying the historical scripts preserved in this repository.
