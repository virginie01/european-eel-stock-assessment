# Reproducibility Notes

## Purpose

This repository preserves the original implementation of the Dynamic Factor Analysis (DFA) conducted as part of a Master's research project completed in 2016.

The objective is to document and archive the methodology, datasets, code, and outputs associated with the research rather than to provide a fully maintained software package.

---

# Historical Context

The files contained in this repository correspond to the implementation of the Dynamic Factor Analysis used during the Master's thesis and subsequent scientific publications.

Following completion of the thesis, the research project evolved and additional analyses were conducted.

Consequently, the archived implementation should be viewed as a historical research snapshot corresponding to the work presented in the thesis rather than the most recent implementation of the analysis.

# Reproducibility Scope

The repository preserves:

- the original recruitment dataset,
- the original R implementation,
- the archived analysis workspace,
- figures produced during the analysis,
- methodological documentation,
- supporting documentation describing assumptions, workflow, and data preparation.

Together, these materials document the complete analytical workflow used during the research project.

---

# Limitations

Because the project was developed more than a decade ago, successful execution on modern systems cannot be guaranteed without modification.

Potential issues include:

- changes in the MARSS package,
- differences among R versions,
- operating system compatibility,
- package dependency changes,
- hard-coded working directory contained in the original script.

The original implementation has intentionally been preserved without modification in order to maintain historical integrity.

# Relationship to Later Research

The Dynamic Factor Analysis documented here represents one component of a broader research programme investigating European eel recruitment dynamics.

Its results informed the subsequent development of the Glass Eel Recruitment Estimation Model (GEREM) and were complemented by environmental correlation analyses presented elsewhere in the thesis.

Readers interested in the broader research framework are encouraged to consult the accompanying GEREM and correlation analysis repositories.

---

# Repository Philosophy

This repository is intended as a documented research archive.

Its purpose is to preserve:

- original datasets,
- original analysis scripts,
- methodological documentation,
- scientific context,
- historical reproducibility.

The repository should therefore be considered a research archive rather than a maintained analytical software package.

Where possible, the original code and datasets have been preserved unchanged and supplemented with documentation describing the analysis and its scientific context.
