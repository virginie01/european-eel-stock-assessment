# DFA_script.R

## Purpose

Main analysis script implementing the Dynamic Factor Analysis.

The script imports the recruitment dataset, preprocesses the observations, fits candidate DFA models using the **MARSS** package, compares competing models using AICc, and saves the resulting analysis workspace.

---

## Main Tasks

The script performs the following steps:

1. Initialize the R environment.
2. Import recruitment time series.
3. Apply data transformations.
4. Select recruitment series for analysis.
5. Produce exploratory figures.
6. Prepare the recruitment matrix.
7. Specify candidate DFA models.
8. Fit models using MARSS.
9. Compare models using AICc.
10. Save the complete R workspace.

---

## Inputs

The script reads:

- `recruitment_series_DFA.csv`

---

## Outputs

The script produces:

- exploratory figures (displayed during execution),
- model comparison table (`model.data`),
- fitted MARSS model objects,
- archived R workspace (`dfa2607.RData`).

---

## External Dependencies

The script requires the following R package:

- MARSS

No custom functions or external scripts are required.

# dfa2607.RData

## Purpose

Archived R workspace generated by `DFA_script.R`.

The workspace preserves the state of the analysis at the end of model fitting.

It may include:

- processed recruitment data,
- fitted MARSS model objects,
- model comparison results,
- intermediate analysis objects,
- figures generated during the analysis.

The workspace is preserved as part of the historical research archive and has not been modified.
