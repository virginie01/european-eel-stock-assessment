# Reproducibility Notes

## Purpose

This repository preserves the original implementation of the Glass Eel Recruitment Estimation Model (GEREM) used during a Master's research project completed in 2016.

The objective is to document and archive the methodology, datasets, model structure, and code associated with the research rather than to provide a fully maintained software package.

---

# Historical Context

The files contained in this repository correspond to a historical implementation of GEREM used during thesis work and subsequent scientific publications.

Following completion of the thesis, the model continued to be developed and applied by other researchers.

Consequently, the version archived here should not be considered the most recent or operational implementation of GEREM.

Instead, it should be viewed as a documented research snapshot corresponding to the work presented in the thesis and associated publications.

---

# Relationship to Later Developments

GEREM continued to evolve after completion of the thesis.

Researchers interested in current operational versions of the model should refer to subsequent publications or contact the research groups responsible for later developments: https://eabx.bordeaux-aquitaine.hub.inrae.fr/qui-sommes-nous/membres/titulaires/drouineau-hilaire

This repository documents the version associated with the thesis and related scientific outputs.

---

# Repository Philosophy

The objective of this archive is preservation, documentation, and transparency.

The repository should be considered a reproducible research archive rather than a maintained software product.

Where possible, original code, data, and model specifications have been preserved in their historical form and accompanied by explanatory documentation.
